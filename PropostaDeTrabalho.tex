\section{Atividades Realizadas}
\label{Proposta de Trabalho:Atividades Realizadas}

A seguir são apresentadas as atividades que já foram realizadas durante o período de mestrado:

\begin{enumerate}
	\item \textbf{Pesquisa de Trabalhos Relacionados} -- Foi realizada uma pesquisa de trabalhos relacionados, a qual foi apresentada no Capítulo \ref{Trabalhos Relacionados};

	\item \textbf{Análise de Hiper Heurísticas} -- Foi conduzida uma análise sobre abordagens hiper-heurísticas aplicadas a outros domínios de problema na literatura. Esta analise foi feita para avaliar qual abordagem tem maior potencial de obter melhores resultados para o PDP.
	
	\item \textbf{Busca por Heurísticas de Baixo Nível} -- Foi conduzida uma busca com o objetivo de encontrar um conjunto de heurísticas de baixo nível para o PDP. Estas  irão compor um \textit{pool} de heurísticas   e foram apresentadas no Capítulo \ref{Metodologia}.
	
	\item \textbf{Criação de Código Fonte Base} -- A criação do código fonte base foi iniciada e já possui as funcionalidades para 	interpretar e avaliar soluções ao PDP. Um interpretador para a gramática apresentada no capítulo \ref{Metodologia} também foi implementado. Mecanismos auxiliares de visualização das conformações também já foram implementados e serão úteis 
\end{enumerate}
